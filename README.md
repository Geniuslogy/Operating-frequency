## Ordlista
<b>Absoluta frekvensvärden</b> - Representerar varje frekvens

<b>Kumulativa frekvensen</b> - Summan av alla absoluta frekvensvärden

<b>Duplicerade frekvensen</b> - Mångfald av samma frekvensvärde
